<h2>Main Features</h2>

<p>Tabular</p>

<ul>
  <li>Ease of usage</li>
  <li>Efficient anonymization (based on pandas DataFrame)</li>
  <li>Numerous anonymization techniques (perturbation, tokenization, synthetic data ...)</li>
</ul>
